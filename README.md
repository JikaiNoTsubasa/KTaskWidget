# KTaskWidget
Widget for kubuntu to manage daily tasks

# Description
Widget to be installed in kubuntu plasma and enabled from the menu "install from local" in plasma.

The widget is a floating icon on the screen that we can move freely from screen to screen and on the screen itself. The icon is always on top of other windows.
When clicking the icon, a small window shows along the icon, which is moved at the same time as the icon.
The window is layered as follow:
- A toolbar with a + icon to create new tasks, a "close" ison to close the widget on the right of the tool bar
- A content composed with a list of tasks with a tickbox to mark the task as done and the name of the task

The app must allow CRUD of tasks

The tasks are saved in a file in the home folder (hidden) and loaded when the widget is enabled or displayed
